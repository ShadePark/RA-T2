## 1. 프로젝트 개요

**ShockGuard All-Weather Resilience Strategy Portfolio**는  
시장 급변(금융위기, 인플레이션, 경기침체, 강세장 전환 등)에 대응하도록 설계된  
**정적(All-Weather) + 동적(Dynamic) 자산배분 결합형 포트폴리오 전략**입니다.

- 목표:  
  - 극단적 시장 쇼크에서의 **MDD 억제**
  - 장기 관점에서의 **안정적 복리 성장**
  - 국면별 자산 전환을 통한 **회복 탄력성 확보**

본 프로젝트는 ETF 기반 글로벌 자산배분 전략을 백테스트, 시각화,  
HTS 연동 자동매매 시스템까지 확장 구현한 **엔드투엔드 퀀트 포트폴리오 프로젝트**입니다.

## 3. 운용 철학

### 정적 자산배분 (All-Weather Core)
- 주식 / 채권 / 금 / 은 / 원자재를 **사전 정의된 비중**으로 장기 보유
- 어떤 경제 국면에서도 포트폴리오 구조가 붕괴되지 않도록 설계

### 동적 자산배분 (ShockGuard Layer)
- 시장 상태를 **카나리아 지표 + 보조 지표**로 판단
- 위험 국면에서는 방어 자산 전면 배치
- 회복·강세 국면에서는 공격 자산 점진적 또는 적극적 진입

### 회복력
- 올웨더 포트폴리오 + 쇼크시 동적 방어 = 회복력 있는 포트폴리오

## 6. 동적 자산배분 로직

### 시장 국면 정의

- **RISK-OFF DEF** : 극도의 공포
- **RISK-OFF INF** : 혼란/인플레이션 공포
- **RECOVERY DEF** : 안전자산 대기
- **RECOVERY INF** : 점진적 진입
- **RISK-ON WEAK** : 약한 상승장
- **RISK-ON STRONG** : 강한 상승장

### 주요 활용 자산
- GLD (금): 모든 위기의 방어 자산
- BIL (단기채): 극단적 위기 시 현금성 완충
- XLE (에너지): 인플레이션 방어
- SPY / QQQ: 회복·강세장 공격 자산
